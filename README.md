
# Intelligent Document Processing System

## Overview
An enterprise-level Intelligent Document Processing System built using OCR, NLP,
Computer Vision, and Streamlit.

## Features
- OCR Extraction
- Resume Parsing
- Invoice Processing
- KYC Verification
- Insurance Document Processing
- PDF Support
- MongoDB Integration
- FastAPI Integration
- Streamlit Dashboard

## Technologies
- Python
- Streamlit
- EasyOCR
- Tesseract OCR
- OpenCV
- spaCy
- MongoDB
- FastAPI

## Installation

pip install -r requirements.txt

## Run Application

streamlit run app.py

## Run FastAPI

uvicorn api.api_service:app --reload
